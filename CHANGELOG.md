# Changelog

All notable changes to Decidr are documented here.
---
## [1.2.1] — 1st Update

### Added
- Added top pill shape bar at top of themes section for easy access.
---
## [1.2.0] — 1st Update

### Added
- Added new Monochrome and Gradient themes.
- Updated the UI.
- Optimized the app.
- Updated the Bottom Navigation bar.
- Animations are now more fluid.
---

## [1.1.0] — 1st Update

### Added
- Added 8 themes under the new customization Tab.
- You can now choose any orb theme from the Customization tab and customize the look of the app instantly.


## [1.0.0] — Initial Release

### Added
- Decision logging with full context capture (situation, options, choice, reasoning)
- Confidence level tagging: Low / Medium / High
- Category system: Career, Finance, Health, Relationships, Personal, Other
- Outcome reviews — revisit past decisions and record what happened
- Insights dashboard with stats and outcome distribution chart
- Review-due badges on decision cards
- Category filter chips on the journal view
- Smooth animated view transitions
- Toast notification system
- Mobile-first responsive layout with safe-area support
- Full offline support via localStorage
- Zero-dependency single-file architecture
