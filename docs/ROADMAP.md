# Roadmap

This document outlines the planned direction for Decidr. Items are loosely ordered by priority.

---

## Near Term

- [ ] **Export** — Download all decisions as a formatted JSON or PDF file
- [ ] **Search** — Full-text search across decision titles and notes
- [ ] **Tags** — Free-form tags in addition to the fixed category system
- [ ] **Spaced repetition scheduling** — Smarter review reminders based on decision age and importance
- [ ] **Edit decisions** — Allow editing a logged decision before any reviews are added

## Medium Term

- [ ] **Optional cloud sync** — Opt-in sync via Google Drive or iCloud (user-controlled, private)
- [ ] **Decision templates** — Pre-filled prompts for common decision types (job offers, purchases, etc.)
- [ ] **Light theme** — A clean light-mode variant
- [ ] **Home screen install prompt** — Nudge users to add to home screen for app-like experience
- [ ] **Share card** — Generate a shareable summary image of a decision (without sensitive details)

## Longer Term

- [ ] **AI reflection** — Optional local AI to surface patterns in your decision history
- [ ] **Decision frameworks** — Built-in prompts for 10/10/10, pre-mortem, second-order thinking
- [ ] **Weekly digest** — Optional email digest of decisions due for review
- [ ] **Import** — Migrate decisions in from plain text or JSON

---

> This is a personal tool. Every feature added is weighed against the principle: *does this make the tool more useful, or just more complex?*
